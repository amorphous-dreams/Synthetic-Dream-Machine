# tests/chats — Archived Lares Behavioral Test Plans

This directory preserves the older Lares HUD / memes / chats behavioral-test era.

These files remain useful as prompt-contract research and exemplar material, but they are no longer the active repository test harness for lararium packages.

## Contents

- `plans/` — v0.2–v0.4 behavioral test plans: gate integrity, register fidelity, frame uncertainty, modes, KAIROS, memory consolidation, etc.
- `expected/` — human-curated chat/HUD exemplar outputs from the old behavioral suite.

## Active test home

Current package and integration test flows live in:

- `packages/*/tests/` for package-local Jest suites.
- `tests/lararium-tw5/sync/` for daemon/CLI sync-decompose-promote flows.
- `tests/bin/` for shared flow runners and helpers.
