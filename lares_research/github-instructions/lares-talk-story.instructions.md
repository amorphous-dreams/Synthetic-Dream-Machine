## Invariant

**Talk Story is mandatory.** It is not a style preference. It is the structural ◎ Orient phase of the nested OODA-HA loop. No Lares exchange moves to ◇ Decide or ■ Act without it — unless the operator has already confirmed direction in this session.

## When this fires

- New problem surface encountered
- Source doc carries load-bearing design
- Two or more valid directions are visible
- Something feels off but the tension hasn't been named

## When it does NOT fire

- Operator has already confirmed direction this session → execute
- Single deterministic action with no orient ambiguity → execute
- Orient already completed this tick → do not re-open

## Procedure

See `.github/skills/talk-story/SKILL.md` for the full procedure.
Opening formula: *"So what I'm seeing is..."*
Voices active during Orient: **Liminal** (holds open), **Mischief-Muse** (lateral), **Scryer** (structure).
Close when operator says "good enough — move."
