# WTF — Wizard Thief Fighter Side Projects

This folder contains Luka Rejec's Wizard Thief Fighter zines and side projects. These files serve as supplemental and inspirational material for the SDM supplement — tone anchors and setting texture rather than mechanical canon.

## Contents

| Folder | Short Name | Description |
|---|---|---|
| [`Eternal_Return_Key/`](Eternal_Return_Key/) | ERK | Eternal Return Key — WTF zine |
| [`There_A_Red_Door/`](There_A_Red_Door/) | TARD | There, A Red Door — WTF zine |
| [`Magitecnica/`](Magitecnica/) | — | Magitecnica Codex 1 & Codex 2 — WTF zines on magitech and powers |

`Eternal_Return_Key/` and `There_A_Red_Door/` each contain one `.md` source file and an `images/` directory. `Magitecnica/` contains two `.md` files (Codex 1 and Codex 2).

## Role in This Repository

These files occupy a supplemental tier in the Lares agent configuration (`builds/agents/Lares_VSCode_Operations.md`):

> `WTF/Eternal_Return_Key/`, `WTF/There_A_Red_Door/`, `WTF/Magitecnica/`: Luka Rejec zines and side projects — supplemental material, not mechanical canon

Lares treats these as tone and setting texture. They inform homebrew synthesis and worldbuilding but do not carry the same mechanical authority as the core SDM source files in `SDM/`.
