# Agentic Wikitext Research Index

This directory collects all primary research, precedent, and protocol notes supporting the memetic sigil and agentic-wikitext specifications. Use this as your entry point for standards alignment, glyph logic, and protocol design.

---

## Research Document Map

- **w3c-spec-research-v01.md**  
  _W3C and adjacent spec precedent for memetic sigils. High-confidence verdicts on lineage, syntax, conformance, and runtime rigor. Use for standards alignment and reviewer-facing arguments._

- **glyph-research.md**  
  _Numeric Character Reference (NCR) research. Syntax, rationale, and examples from Wikipedia/W3C. Use for Unicode and NCR triplicate logic._

- **sigil-research.md**  
  _Philosophical and functional analysis of sigils, including "mu" and Maybe Logic. Explores the shape/motion metaphor and the role of sigils in agentic wikitext._

- **memetic_sigils_v4_draft.md**  
  _Current working draft of the memetic sigil specification. All research docs feed into this spec._

- **memetic-sigils_v3.md**  
  _Previous version of the memetic sigil codeset. Useful for migration and historical comparison._

- **mana-manao-model.md**  
  _Mana (runtime potency) and Manaʻo (intent confidence) model. Defines the entangled rating system for agentic objects._

- **unified-attention-scale.md**  
  _Defines the invariant sigils and scale bands for attention range (time, text, space)._

- **unified-five-band-v01.md**  
  _Canonical five-band chronometer and meme architecture. Maps scale slots, phase glyphs, and fragment format._

- **LOCI.md**  
  _Wild Magic and symbolic umwelt engineering. Explains the foundational metaphors and transclusion grammar._
